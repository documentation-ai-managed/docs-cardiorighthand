## Prerequisites

<Callout kind="info">

Before starting, ensure you have:
- A valid email address for account creation
- Access to a modern web browser (Chrome, Firefox, or Safari)
- Basic familiarity with web dashboards

No special software required.

</Callout>

## Create Account and Log In

Follow these steps to gain access to CardioRightHand.

<Steps>
  <Step title="Visit Dashboard" icon="monitor">
    Open your browser and navigate to `https://dashboard.example.com`.
  </Step>
  <Step title="Sign Up" icon="user-plus">
    Click **Sign Up**. Enter your email, create a strong password, and submit the form.
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your inbox for a verification email from CardioRightHand. Click the link to activate your account.
  </Step>
  <Step title="Log In" icon="log-in">
    Return to the dashboard, enter your credentials, and click **Log In**. You now access the main interface.
  </Step>
</Steps>

## Interact with Your First Object

Start by viewing an existing object or creating a new one.

<Tabs>
  <Tab title="View Object CRH-001" icon="eye">
    <Steps>
      <Step title="Search Objects" icon="search">
        In the dashboard, go to **Objects** > **Search**. Enter `CRH-001`.
      </Step>
      <Step title="Review Details" icon="file-text">
        Examine containment procedures, description, and addendums. Note the object's anomalous effect: induces cardiac anomalies in observers.
      </Step>
    </Steps>

  <Tab title="Create New Object" icon="plus">

      <Step title="New Object" icon="file-plus">

      </Step>
    </Steps>
  </Tab>
</Tabs>

## Access Basic Zones

Zones control object containment. Begin with low-level access.

<CodeGroup tabs="cURL,JavaScript">
  ```bash
  curl -X GET https://api.example.com/v1/zones \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json"
  ```
  ```javascript
  const response = await fetch('https://api.example.com/v1/zones', {
    method: 'GET',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    }
  });
  const zones = await response.json();
  console.log(zones); // Lists Zone-1A, Zone-2B, etc.
  ```
</CodeGroup>

<Callout kind="tip">

Start with `Zone-1A` (safe testing area). Higher zones require clearance.

</Callout>
